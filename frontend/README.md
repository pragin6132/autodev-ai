# AutoDev AI - Frontend Application

This directory contains the user interface dashboard for monitoring and managing the AutoDev AI autonomous agent platform.

---

## Technical Architecture

- **Vanilla HTML5 & ES6 Javascript**: Lightweight and highly responsive UI.
- **Vanilla CSS3**: Layout styling with HSL variables, dark mode styling, and custom animations.
- **Vite**: Ultra-fast module bundling tool.

---

## Development Setup

1. Ensure Node.js 20+ is installed.
2. Install npm packages:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Build for deployment:
   ```bash
   npm run build
   ```
