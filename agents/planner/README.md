# Planner Agent

The **Planner Agent** designs execution blueprints for AutoDev AI.

---

## Duties & Responsibilities
1. **Analysis**: Deciphers high-level system requirements into specific codebase changes.
2. **Task Generation**: Writes `implementation_plan.md` and compiles the initial `task.md` TODO list.
3. **Refinement**: Modifies plans based on user corrections or failure feedback from developer agents.
