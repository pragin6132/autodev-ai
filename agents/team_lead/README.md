# Team Lead Agent

The **Team Lead Agent** coordinates the execution loop of AutoDev AI.

---

## Duties & Responsibilities
1. **Intake & Discovery**: Interacts with the user, parses requirements, and accesses project directory files.
2. **Delegation**: Commands the Planner agent to create task steps, and assigns steps to developer agents.
3. **Audit**: Verifies quality, checks test outputs, and requests user approval prior to final merge operations.
